# Portfolio

Personal portfolio showcasing work in systems programming, language design, and infrastructure. Features a project showcase, technical blog, and background information about low-level software development. This is a port of the SvelteKit original to Jane Street's Bonsai, compiled to JavaScript and served as a client-rendered single-page app.

## Tech Stack

- **Bonsai** - Incremental UI and routing
- **OCaml** - Type safety
- **js_of_ocaml** - OCaml-to-JavaScript compilation
- **omd** - Markdown blog content compiled at build time
- **dune** - Build tool and dev server
- **just** - Command runner
